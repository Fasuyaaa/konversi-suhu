# konversi-suhu

Terdapat beberapa satuan suhu yang bisa di konversi:
- Celsius
- Fahrenheit
- Reamur
- Kelvin

----------------------------------------------------------------------------------

# INSTALASI

#Cara install di LINUX
1. Buka terminal
2. Ketik "sudo apt-get install python3"
3. Ketik "git clone https://github.com/Fasuyaaa/konversi-suhu.git"
4. cd konversi-suhu 
5. ketik "python3 suhu.py"
